
# Todo List

A simple Todo List application built with React that allows users to add and delete tasks easily.

## Features

* Add new tasks
* Delete existing tasks
* Clean and user-friendly interface
* Built using React functional components and state management

## Technologies Used

* React.js
* JavaScript
* HTML
* CSS

## Getting Started

### Prerequisites

Make sure you have Node.js and npm installed.

### Installation

1. Clone the repository:

```bash
git clone https://github.com/Ishita3104/todo-list.git
```

2. Navigate to the project directory:

```bash
cd todoList
```
3. Navigate to the project folder:

```bash
cd todo
```

4. Install dependencies:

```bash
npm install
```

5. Run the application:

```bash
npm start
```

The app will open in your browser at:

```bash
http://localhost:3000
```

## How to Use

1. Type a task into the input field.
2. Click the **Add** button to add it to the list.
3. Click the **Delete** button next to a task to remove it.

## Project Purpose

This project was created to practice React fundamentals, including:

* Components
* JSX
* State management using `useState`
* Event handling
